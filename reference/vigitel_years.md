# List available VIGITEL survey years

Returns a vector of years for which VIGITEL microdata is available for
download from the Ministry of Health website.

## Usage

``` r
vigitel_years()
```

## Value

An integer vector of available years (2006-2024).

## Examples

``` r
vigitel_years()
#>  [1] 2006 2007 2008 2009 2010 2011 2012 2013 2014 2015 2016 2017 2018 2019 2020
#> [16] 2021 2022 2023 2024
```
